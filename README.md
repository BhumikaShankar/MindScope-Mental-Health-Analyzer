# MindScope - Mental Health Assessment and Support System

MindScope is an AI-powered web application that provides preliminary mental health assessment and personalized emotional support. The system evaluates symptoms related to **Depression, Anxiety, OCD, and PTSD** through a questionnaire and intelligently selects the most appropriate counselling approach using a machine learning routing model. :contentReference[oaicite:0]{index=0}

## Features

-  User Registration & Login
-  Mental Health Questionnaire
-  An Intermediate Routing ML Model based on Logisitic Regression to navigating to the most suitable engine
-  ML-based Counselling Engine Selection
-  Rule-Based, Retrieval-Based & Generative AI Chat Support
-  SQLite Database Integration

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript, Bootstrap
- **Backend:** Flask (Python)
- **Database:** SQLite, SQLAlchemy
- **AI/ML:** Scikit-learn, Sentence Transformers, TensorFlow/Keras, Hugging Face Transformers

## Installation

```bash
git clone https://github.com/BhumikaShankar/MindScope-Mental-Health-Analyzer.git

cd MindScope-Mental-Health-Analyzer

pip install -r requirements.txt

python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```
